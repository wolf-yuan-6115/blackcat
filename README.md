## Setup

1. Install Node.js 18 or newer, and `pnpm` package manager
2. Create `config.json`, with following content:
  ```json
  {
    "token": "Your Discord bot token"
  }
  ```
3. Run `pnpm install`
4. Run `pnpm start` to start
